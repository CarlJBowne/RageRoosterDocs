AreaRegistry

A global registry asset of all AreaAssets in the project.
Used to access AreaAssets at runtime by name or all at once.

Base Type: SingletonAsset<AreaRegistry>

Static Methods

GetArea(string)

Returns: AreaAsset

Get an area in the registry by name.
  • Parameters
  • string : name

GetAll()

Returns: AreaAsset[]

Returns All areas in the registry.

Editor_AddArea( AreaAsset)

Adds an AreaAsset to the registry asset. EDITOR ONLY.
  • Parameters
  • AreaAsset : area
    The AreaAsset to add to the registry asset.
